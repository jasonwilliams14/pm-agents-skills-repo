# Success Metrics Guide

## Purpose
Define measurable success criteria that validate product goals and user outcomes.

## Metric Categories

### 1. Usage Metrics
Measure adoption and engagement

**Examples:**
- Daily/Weekly/Monthly Active Users (DAU/WAU/MAU)
- Feature adoption rate
- Time spent in product
- Number of actions/operations performed
- Return user rate

### 2. Performance Metrics
Measure technical excellence

**Examples:**
- Response time / Latency (p50, p95, p99)
- Throughput (requests per second)
- Error rate
- Uptime / Availability (SLA)
- Resource utilization

### 3. Business Impact Metrics
Measure business value

**Examples:**
- Revenue impact
- Cost savings
- Time saved per user
- Conversion rates
- Customer retention

### 4. User Satisfaction Metrics
Measure user sentiment

**Examples:**
- NPS (Net Promoter Score)
- CSAT (Customer Satisfaction)
- Support ticket volume
- User feedback scores
- Feature request volume
