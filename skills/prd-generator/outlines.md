# PRD Template Router

Analyze the completed 'Requirements Extraction Engine' metadata and select exactly ONE matching template route below.

## 1. Route: Standard Feature
- **Criteria:** High business/user logic, user stories, customer-facing value.
- **Action:** Load and apply `assets/templates/standard-template.md`.

## 2. Route: Technical Infrastructure
- **Criteria:** Backend, API updates, database migrations, schema definitions, zero UI impact.
- **Action:** Load and apply `assets/templates/technical-template.md`.

## 3. Route: UX-Focused Interaction
- **Criteria:** Major frontend overhaul, design-system tokens, complex interactive states, accessibility upgrades.
- **Action:** Load and apply `assets/templates/ux-template.md`.
