# AI Gateway Design Workflow

1. Identify:
   - model providers
   - routing requirements
   - auth model
   - token tracking

2. Evaluate:
   - latency
   - caching
   - retries
   - streaming support

3. Verify:
   - observability
   - rate limiting
   - prompt inspection
   - tenant isolation

4. Generate:
   - architecture diagram
   - traffic flow
   - deployment model
