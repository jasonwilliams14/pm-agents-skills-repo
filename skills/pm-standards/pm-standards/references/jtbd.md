# Jobs-to-be-Done (JTBD) Framework

Use this framework for discovery and defining user needs.

## The JTBD Formula
"When I [situation], I want to [motivation], so I can [expected outcome]."

## Discovery Guidelines
- Focus on the progress the user is trying to make.
- Identify the "job" the user is "hiring" the product to do.
- Analyze the "Forces of Progress":
  - **Push:** Problems with the current solution.
  - **Pull:** Attraction of the new solution.
  - **Anxiety:** Fear of the new solution.
  - **Allegiance:** Attachment to the current solution.
